# Limited Run AW 2025 Badge

### Build List Items
| Reference | Value | Footprint |
| --- | --- | --- |
| C1 | 0.01uF | 0603 |
| C2 | 10uF | 0805 |
| D1-D6 | White | 0603 |
| R1 | 50kΩ | 0805 |
| R2 | 20kΩ | 0805 |
| R3, R4 | 56Ω | 0805 |
| U1 | 555 Timer | SOIC-8 |

------
### Steps
1. Solder Timer
2. Solder Capacitors & Resistors
3. Solder LEDs
4. Bridge Jumper to obtain flashing or solid-ish affect.
5. [OPTIONAL] Apply hot glue to difuse light
